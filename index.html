Política de Privacidade - SyncSchool

Última atualização: 6 de março de 2026

1. Finalidade

O SyncSchool é uma extensão com propósito único: sincronizar favoritos do Google Chrome entre um computador Pai e computadores Filhos.

2. Dados processados

A extensão pode processar apenas os dados necessários para a sincronização:

  • Favoritos do navegador (título, URL, pastas e ordem).
  • Configurações locais da extensão (papel Pai/Filho, URL do servidor, canal, token opcional, intervalo).
  • Informações técnicas de conexão usadas para sincronização.

3. Uso dos dados

Os dados são usados exclusivamente para o funcionamento da sincronização de favoritos. O SyncSchool não vende dados e não usa dados para publicidade.

4. Compartilhamento

Não há compartilhamento de dados com terceiros para fins não relacionados ao funcionamento da extensão.

5. Armazenamento

As configurações ficam no armazenamento local da extensão no Chrome. Os dados sincronizados trafegam apenas para o servidor configurado pelo próprio usuário.

6. Segurança

Recomenda-se uso de token de autenticação e rede confiável. A segurança do servidor configurado é de responsabilidade do administrador do ambiente.
